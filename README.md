

Esta es mi solución para **Minimal Blog Card** de DevChallenges.io

El desafío consistía en construir una tarjeta de blog que se pareciera lo más posible al diseño proporcionado, respetando las medidas, colores y tipografía.

[Desktop Preview](./assets/img/screenshot.png)


### Built with
- HTML5 
- CSS3 (Flexbox, Custom Properties)
- Google Fonts (Sora)
- Git & GitHub

### What I learned
En este proyecto puse en práctica mis conocimientos fundamentales de HTML y CSS. Algunos aprendizajes clave fueron:

- Dominio de Flexbox: Comprendí mejor el comportamiento de los ejes y cómo alinear elementos eficientemente usando justify-content, align-items y flex-direction.

- Entendí la importancia de box-sizing: border-box para controlar el padding sin alterar el ancho total de los componentes.

- Aprendí a estilizar y alinear elementos inline-block (como la etiqueta "Design") dentro de un contenedor flexible.
